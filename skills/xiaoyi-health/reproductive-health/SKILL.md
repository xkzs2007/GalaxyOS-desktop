# 生殖健康数据获取与分析指南

## 一、数据获取策略

### 场景 A：周期预测与当前阶段

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_menstrual_cycle --date today
```

> `get_menstrual_cycle` 返回数据自动包含未来 30 天的预测经期（`predictedPeriodStartDate/EndDate`）和易孕窗口（`fertileStartDate/EndDate`），无需额外调用。

### 场景 B：规律性评估（建议 3-6 个月数据）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_menstrual_cycle --last-days 90
```

### 场景 C：排卵追踪（BBT + 周期联合）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"name":"get_menstrual_cycle","args":{"date":"today"}},
  {"name":"get_body_temperature","args":{"startDate":"YYYY-MM-DD","endDate":"YYYY-MM-DD"}}
]'
```

### 场景 D：症状管理（PMS + 情绪 + 睡眠）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"name":"get_menstrual_cycle","args":{"date":"today"}},
  {"name":"get_emotion","args":{"date":"today"}},
  {"name":"get_sleep","args":{"date":"today"}}
]'
```

## 二、分析框架

### 第一步：问题分类

| 用户说 | 问题类型 | 取数方案 |
|--------|---------|---------|
| "我下次月经什么时候来？" | **周期预测** | `get_menstrual_cycle`（含预测数据）|
| "我的周期规律吗？" | **规律性评估** | 3-6 个月数据 |
| "我痛经 / 有经前症状" | **症状管理** | 当前周期阶段 + 情绪 + 睡眠 |
| "我在排卵吗？" | **排卵追踪** | 体温双相模式 + 周期天数 |
| "经期应该怎么运动？" | **分阶段指导** | 当前周期阶段 |

### 第二步：周期参考标准

| 指标 | 正常范围 |
|------|---------|
| 周期长度 | 22-35 天 |
| 经期持续时间 | 3-7 天 |
| 排卵后体温升高 | ≥ 0.3°C |
| 经前体重波动 | +0.5-2 kg（水分潴留，属正常）|

### 第三步：月经周期四个阶段

| 阶段 | 时间（28 天周期） | 身体状态 | 运动建议 |
|------|-----------------|---------|---------|
| **月经期** | 第 1-5 天 | 精力偏低，可能痛经 | 倾听身体，轻量运动完全可以 |
| **卵泡期** | 第 6-13 天 | 精力增强，雌激素升高 | 高强度训练的最佳时机 |
| **排卵期** | 约第 14 天 | 体能巅峰，韧带松弛度略增 | 巅峰表现，但受伤风险稍高 |
| **黄体期** | 第 15-28 天 | 易疲劳，情绪波动，体温升高 | 降低强度，以中等活动为主 |

### 第四步：规律性评估

基于 3-6 个月数据：
- 变异 < 3 天：非常规律
- 变异 3-7 天：中等规律
- 变异 > 7 天：不规律，建议调查原因

**常见不规律原因**：压力、体重显著变化、过度运动、旅行、生病

### 第五步：BBT 分析

- **双相模式**（排卵前低、排卵后高 ≥0.3°C）：表明已排卵
- **单相模式**（无明显变化）：可能为无排卵周期
- 排卵后体温持续 11 天以上后下降 → 月经即将来临

### 第六步：PMS 症状管理

常见 PMS 症状（黄体期，约第 21-28 天）：
- 情绪变化：烦躁、焦虑、情绪低落（孕酮波动所致）
- 身体症状：腹胀、乳房胀痛、头痛
- 食欲增加：嗜碳水化合物（激素作用，不是缺乏意志力）

循证缓解方法：
- 经前 5-7 天减少钠和咖啡因摄入
- 整个周期保持规律的中等强度运动可减轻 PMS
- 富含镁的食物：黑巧克力、坚果、深绿色叶菜

### 沟通原则

始终保持尊重、温暖、不带评判的语气：
- **不好**："你的周期不规律，这可能说明有问题。"
- **好**："你过去 4 个月的周期在 25 到 33 天之间波动。一定程度的变化是正常的，可在下次妇科检查时提一下。"

### 红线

| 信号 | 处理方式 |
|------|---------|
| 周期持续 <22 天或 >35 天 | 建议咨询妇科医生 |
| 经期持续 >7 天或经量过大 | 建议就医评估 |
| 闭经（非怀孕，停经 3 个月以上）| 建议妇科评估 |
| 严重痛经影响日常生活 | 建议就医，有效治疗方案很多 |
| 用户询问备孕/生育规划 | 建议咨询妇科医生，无法替代专业生育评估 |
