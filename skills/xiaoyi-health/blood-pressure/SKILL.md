# 血压数据 CLI 获取指南

## 时间查询规范

**趋势分析默认范围**：单次血压读数受咖啡因、压力、测量时间等因素影响，不可单独作为判断依据。趋势分析时默认取最近 **7-30 天**，以多次均值评估血压状态。用户未指定时间范围时，优先取最近 7 天作为起点。

## 命令示例

### 获取今日血压
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_pressure --date today
```

### 获取指定日期血压
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_pressure --date 2024-01-15
```

### 获取最近 7 天血压趋势
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_pressure --last-days 7
```

### 获取最近 30 天血压趋势
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_pressure --last-days 30
```

### 获取指定日期范围血压数据
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_pressure --start-date 2024-01-01 --end-date 2024-01-31
```

## 返回字段说明

| 字段 | 说明 |
|------|------|
| `systolic` | 收缩压（mmHg，俗称"高压"） |
| `diastolic` | 舒张压（mmHg，俗称"低压"） |
| `readings[]` | 全天各次测量记录 |

## 参考范围（成人）

| 分类 | 收缩压 | 舒张压 |
|------|--------|--------|
| 正常 | <120 | <80 |
| 偏高 | 120–139 | 80–89 |
| 高血压 I 级 | 140–159 | 90–99 |
| 高血压 II 级 | ≥160 | ≥100 |

## 分析框架

### 血压分级（中国高血压防治指南）

**始终使用多次测量的平均值，而非单次读数。**

| 类别 | 收缩压 (mmHg) | 舒张压 (mmHg) |
|------|--------------|--------------|
| 理想 | < 120 | < 80 |
| 正常 | 120-129 | 80-84 |
| 正常高值 | 130-139 | 85-89 |
| 1 级高血压（轻度） | 140-159 | 90-99 |
| 2 级高血压（中度） | 160-179 | 100-109 |
| 3 级高血压（重度） | ≥ 180 | ≥ 110 |

当收缩压和舒张压属于不同级别时，以较高的级别为准。

### 脉压分析

脉压 = 收缩压 - 舒张压

| 范围 | 解读 |
|------|------|
| 30-60 mmHg | 正常 |
| > 60 mmHg | 可能提示动脉硬化 |
| < 30 mmHg | 可能提示心输出量减少 |

### 血压模式

- **昼夜节律**：正常情况下睡眠期间血压下降 10-20%（杓型），非杓型心血管风险更高
- **白大褂高血压**：诊室高但家中正常，家庭监测可排除
- **隐蔽性高血压**：诊室正常但家中偏高，危险性更高

### 生活方式影响因素

| 因素 | 对血压的影响 |
|------|-------------|
| 每日盐摄入 > 6g | 显著升高 |
| 每减重 5 kg | 约降低 5 mmHg |
| 规律运动（每周 150 分钟中等强度） | 降低 5-8 mmHg |
| 长期压力 / 睡眠不佳 | 持续升高 |
| 富钾食物（香蕉、菠菜） | 有助于抵消钠的影响 |

### 沟通原则

不要因单次读数而惊慌，始终引用多日平均值。对正常高值用户强调这是生活方式改变最有效的阶段。

### 红线

| 信号 | 处理方式 |
|------|---------|
| 多次测量收缩压持续 > 160 mmHg | 建议预约医生正规检查 |
| 血压 > 180/110 | 高血压危象范围，伴症状立即就医 |
| 询问调整或停用降压药 | 药物调整必须由医生决定，突然停药有风险 |
| 剧烈头痛、视力变化、胸痛伴血压升高 | 立即就医或拨打急救电话 |
