# 血糖教练 CLI 获取与分析指南

## 一、数据获取策略

### 基线检查（今日血糖）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_glucose --date today
```

### 趋势分析（近期血糖 + 饮食）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"tool":"get_blood_glucose","args":{"date":"today"}},
  {"tool":"get_nutrition","args":{"date":"today"}}
]'
```

### 食物-血糖关联分析（指定日期）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"tool":"get_blood_glucose","args":{"date":"2024-01-15"}},
  {"tool":"get_nutrition","args":{"date":"2024-01-15"}}
]'
```

### 全面评估（近 7 天血糖 + 营养趋势）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"tool":"get_blood_glucose","args":{"date":"today"}},
  {"tool":"get_nutrition","args":{"last_days":7}}
]'
```

## 返回字段说明

| 工具 | 字段 | 说明 |
|------|------|------|
| `get_blood_glucose` | `value` | 血糖值（mmol/L） |
| `get_blood_glucose` | `readings[]` | 全天各时段测量记录 |
| `get_nutrition` | `calories` | 总摄入卡路里（千卡） |
| `get_nutrition` | `carbs` | 碳水化合物（克） |
| `get_nutrition` | `protein` | 蛋白质（克） |
| `get_nutrition` | `fat` | 脂肪（克） |
| `get_nutrition` | `meals[]` | 各餐饮食记录 |

## 二、分析框架

### 问题分类

| 用户说 | 问题类型 | 数据策略 |
|--------|---------|---------|
| "我的血糖正常吗？" | 基线检查 | `get_blood_glucose(today)` |
| "什么食物让我血糖飙升？" | 食物-血糖关联 | 血糖 + 同日营养记录 |
| "我有糖尿病风险吗？" | 风险评估 | 近期空腹血糖趋势 |
| "血糖低了，感觉发抖" | 低血糖 | 当前读数 + 近期饮食 |
| "运动对血糖有什么影响？" | 运动-血糖关联 | 运动前后血糖对比 |

### 血糖分级

| 指标 | 正常 | 糖尿病前期 | 糖尿病 |
|------|------|-----------|--------|
| 空腹血糖 (mmol/L) | 3.9–6.1 | 6.1–7.0 | ≥ 7.0 |
| 餐后 2 小时 (mmol/L) | < 7.8 | 7.8–11.1 | ≥ 11.1 |

**核心原则**：糖尿病前期通过生活方式改变是**可逆的**，这是关键的干预窗口期。

### 升糖指数（GI）参考

| GI 类别 | 范围 | 示例 |
|---------|------|------|
| 低 GI | < 55 | 燕麦、糙米、豆类、大多数蔬菜、苹果 |
| 中 GI | 55–70 | 全麦面包、甜玉米、香蕉 |
| 高 GI | > 70 | 白米饭、白面包、土豆泥、含糖饮料 |

> GI 受搭配食物、烹饪方式、食用量及个体差异影响，血糖负荷比单独 GI 更重要。

### 控糖策略

**进食顺序效应（有循证支持）**：
1. 先吃蔬菜
2. 再吃蛋白质和脂肪
3. 最后吃碳水化合物
→ 可减少餐后血糖飙升 20–30%

**餐后步行**（最有效的单一干预）：
- 餐后 30 分钟内步行 15–30 分钟，可减少餐后血糖飙升 1–2 mmol/L
- 即使步行 10 分钟也有帮助

**进餐规律**：
- 规律进餐时间有助于稳定血糖
- 避免跳餐（引发代偿性暴食和更大血糖波动）
- 深夜进食血糖反应更高（昼夜节律性胰岛素抵抗）

### 低血糖识别

| 血糖水平 | 严重程度 | 症状 |
|----------|---------|------|
| 3.9–3.3 mmol/L | 轻度 | 颤抖、出汗、饥饿、焦虑 |
| 3.3–2.8 mmol/L | 中度 | 意识混乱、视力模糊 |
| < 2.8 mmol/L | 重度 | 意识丧失 — 紧急情况 |

**15 法则**（轻中度低血糖）：摄入 15g 快速碳水，等待 15 分钟，复查血糖。

### 血糖 + 营养联合分析

将营养数据与血糖读数对应，识别：
- **问题食物**：哪顿饭后血糖飙升明显
- **安全食物**：哪些食物组合血糖平稳
- **碳水总量**：当日碳水是否过高导致持续偏高

示例洞察："昨天吃白米饭后血糖到 10.2，前天吃糙米只到 7.6。换成糙米或减少白米饭的量，能产生实际效果。"

### 跨域分析

- **血糖 + 运动**：有氧运动即时降糖；抗阻训练改善长期胰岛素敏感性
- **血糖 + 睡眠**：睡眠不足（< 6 小时）显著增加胰岛素抵抗，即使 1–2 天也有影响
- **血糖 + 压力**：长期压力 → 皮质醇 → 空腹血糖升高
- **血糖 + 体重**：内脏脂肪是胰岛素抵抗最强预测因素，减重 5% 即可显著改善血糖

### 沟通原则

赋能而非危言耸听：

- **不好**："你的空腹血糖 6.3，属于糖尿病前期范围。"
- **好**："这周你的空腹血糖平均 6.3 mmol/L — 在糖尿病前期范围，但重要的是：这个阶段对生活方式改变反应非常好，很多人通过饮食和运动调整把指标恢复正常。让我们看看什么方法对你最有帮助。"

### 红线

| 信号 | 处理方式 |
|------|---------|
| 空腹血糖持续 ≥ 7.0 mmol/L | 建议就医评估糖尿病诊断 |
| 餐后血糖 > 11.1 mmol/L | 糖尿病范围，建议就医沟通 |
| 低血糖症状（颤抖、意识混乱） | 立即摄入快速糖分；15 分钟未改善则就医 |
| 询问调整或停用降糖药 | 药物调整必须由医生决定 |
| 重度低血糖（< 2.8 或意识丧失） | 医疗紧急情况，立即拨打急救电话 |
