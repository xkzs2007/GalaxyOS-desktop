# 压力数据获取与分析指南

## 一、数据获取策略

### 场景 A：当日压力评估

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"name":"get_stress","args":{"date":"today"}},
  {"name":"get_heart_rate","args":{"date":"today"}},
  {"name":"get_sleep","args":{"date":"today"}}
]'
```

### 场景 B：慢性压力 / 职业倦怠趋势分析

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"name":"get_stress","args":{"startDate":"YYYY-MM-DD","endDate":"YYYY-MM-DD"}},
  {"name":"get_hrv","args":{"startDate":"YYYY-MM-DD","endDate":"YYYY-MM-DD"}},
  {"name":"get_sleep","args":{"startDate":"YYYY-MM-DD","endDate":"YYYY-MM-DD"}},
  {"name":"get_emotion","args":{"startDate":"YYYY-MM-DD","endDate":"YYYY-MM-DD"}}
]'
```

或便捷参数：
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_stress --last-days 7
node ./skills/xiaoyi-health/bin/pha-claw.js get_hrv --last-days 7
```

### 数据采集决策表

| 关注点 | 取数方案 |
|-------|---------|
| 急性压力反馈 | `get_stress` + `get_heart_rate` + `get_sleep`（今日） |
| 数据驱动查询 | 同上 |
| 慢性压力/倦怠 | 以上全部（范围查询，7-14 天） |
| 技巧请求 | 可选，用户只需要建议时不必取数 |

## 二、分析框架

### 第一步：分类压力关注点

| 用户描述 | 关注类型 | 调查方向 |
|---------|---------|---------|
| "我压力好大" | **急性压力** | 先认可，用数据确认信号 |
| "我很焦虑 / 快崩溃了" | **情绪困扰** | 先倾听，数据其次 |
| "我有压力吗？" | **数据驱动** | 心率 + 睡眠 + HRV 综合分析 |
| "帮我放松" / "怎么减压？" | **技巧请求** | 提供可操作方法 |
| "我快燃尽了" | **慢性压力** | 多日趋势，建议专业支持 |

### 第二步：压力等级评估

设备压力评分（1-99）：

| 等级 | 分值 | 状态 | 应对 |
|------|------|------|------|
| 放松 | 1-29 | 副交感神经占优 | 肯定良好恢复状态 |
| 正常 | 30-59 | 适度压力，平衡 | 无需干预 |
| 中等 | 60-79 | 压力升高，需关注 | 建议具体减压技巧 |
| 偏高 | 80-99 | 高压，交感神经过激 | 建议停止当前活动，呼吸练习 |

### 第三步：生理压力指标（无直接评分时）

**心率信号**：
- 静息心率高于基线 10+ bpm → 很可能有压力
- 静息心率高于基线 5-10 bpm → 可能有压力

**HRV 信号**：
- HRV 低于个人基线 3 天以上 → 身体承受累积压力
- 低 HRV + 高压力评分 = 强力生理确认

**睡眠信号**：
- 入睡延迟（就寝时间比平时晚很多）→ 可能在反复思考
- 连续 2 晚睡眠 < 6 小时 → 压力相关睡眠紊乱
- 早醒 → 常见焦虑症状

### 第四步：多信号综合评估

| 心率 | 睡眠 | 活动 | 评估 |
|------|------|------|------|
| 升高 | 差 | 下降 | **压力迹象明显** |
| 升高 | 差 | 正常 | **中等压力** |
| 正常 | 差 | 正常 | **可能有压力**（心理压力未反映在心率上）|
| 正常 | 正常 | 正常 | **较低**（数据不支持，但认可用户感受）|

> 重要：没有数据异常不代表没有压力。始终认可用户的主观感受。

### 第五步：职业倦怠风险检测

5 天预警阈值——以下情况全部持续 5 天以上：
- 压力评分平均 ≥60
- HRV 持续下降
- 睡眠质量恶化
- 活动水平下降

### 第六步：即时缓解技巧

**呼吸练习（有循证支持）**：
- **4-7-8 呼吸法**：吸气 4 秒，屏住 7 秒，呼气 8 秒，重复 4 个循环
- **箱式呼吸法**：吸气 4 秒，屏住 4 秒，呼气 4 秒，屏住 4 秒
- **生理叹息法**：通过鼻子双重吸气，然后通过嘴慢慢呼气（最快的单次减压动作）

**快速技巧（5 分钟以内）**：
- **5-4-3-2-1 接地法**：5 样看到的、4 样触摸到的、3 样听到的、2 样闻到的、1 样尝到的
- **冷水刺激手腕/面部**：触发潜水反射，降低心率
- **渐进式肌肉放松**：依次绷紧各组肌肉 5 秒，放松 10 秒

**日常习惯（最强证据）**：
- 规律的中等强度运动（每周 150 分钟）
- 睡眠规律性
- 下午 2 点后限制咖啡因

### 沟通原则

先认可，再建议：
- **不好**："试试呼吸练习，去散个步。"
- **好**："这听起来真的很不容易。让我看看你的数据……[数据分析]。这里有个方法可能现在就有帮助：[具体技巧]。"

**绝对不要说**："放轻松" / "没那么严重" / "每个人都有压力"
