# 体温教练 CLI 获取与分析指南

## 一、数据获取策略

### 今日体温检查

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_body_temperature --date today
```

### 体温 + 经期联合分析（近 30 天）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"tool":"get_body_temperature","args":{"last_days":30}},
  {"tool":"get_menstrual_cycle","args":{"last_days":30}}
]'
```

### BBT 双相分析（指定周期范围）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"tool":"get_body_temperature","args":{"start_date":"2024-01-01","end_date":"2024-01-31"}},
  {"tool":"get_menstrual_cycle","args":{"start_date":"2024-01-01","end_date":"2024-01-31"}}
]'
```

### 急性体温关注（今日体温 + 睡眠交叉验证）

```bash
node ./skills/xiaoyi-health/bin/pha-claw.js --tools '[
  {"tool":"get_body_temperature","args":{"date":"today"}},
  {"tool":"get_sleep","args":{"date":"today"}}
]'
```

## 返回字段说明

| 工具 | 字段 | 说明 |
|------|------|------|
| `get_body_temperature` | `current` | 当前体温（℃） |
| `get_body_temperature` | `avg` | 当日平均体温（℃） |
| `get_body_temperature` | `max` / `min` | 当日最高/最低体温（℃） |
| `get_body_temperature` | `readings[]` | 全天各时段体温记录 |
| `get_menstrual_cycle` | `phase` | 当前周期阶段 |
| `get_menstrual_cycle` | `cycleDay` | 当前周期第几天 |
| `get_menstrual_cycle` | `nextPeriod` | 预计下次月经日期 |

## 二、分析框架

### 问题分类

| 用户说 | 问题类型 | 数据策略 |
|--------|---------|---------|
| "我的体温正常吗？" | 基线检查 | 体温 vs 个人基线 |
| "我好像发烧了" | 急性升高 | 今日体温 + 睡眠/压力背景 |
| "我最近体温一直偏高" | 趋势关注 | 近 30 天体温趋势 |
| "我的体温随月经周期怎么变化？" | 周期相关性 | 体温 + 经期双相分析 |
| "能通过体温确认排卵吗？" | 排卵确认 | BBT 双相模式验证 |

### 体温分级标准

**始终先建立个人基线**（分析近 30 天数据），再与人群标准对比。

| 分类 | 范围（腋下） | 说明 |
|------|------------|------|
| 低体温 | < 36.0°C | 需调查代谢、循环或甲状腺问题 |
| 正常 | 36.0–37.2°C | 健康范围 |
| 低热 | 37.3–38.0°C | 监测；常因运动、压力、脱水引起 |
| 中度发热 | 38.1–39.0°C | 可能生病；建议休息并持续监测 |
| 高热 | > 39.0°C | 建议就医评估 |

> **可穿戴设备提示**：腕部皮肤温度与腋下/口腔测量值不同，误差约 ±0.3°C，解读时需说明。

### BBT 双相分析（女性月经周期）

体温在女性月经周期中呈典型双相模式：

| 阶段 | 时间（28 天周期） | 体温模式 |
|------|-----------------|---------|
| 卵泡期 | 第 1–14 天（月经后到排卵） | 基线偏低（约 36.2–36.5°C） |
| 黄体期 | 第 15–28 天（排卵后） | 升高 0.3–0.5°C，持续到下次月经 |

**关键洞察**：
- 排卵后体温升高 ≥0.3°C 可**确认已排卵**
- 黄体期体温偏高是正常且预期的，不需要担心
- 如无双相模式（体温始终平稳），可能需要进一步调查

### 联合分析：体温 + 经期

将体温数据与经期阶段对照：

| 情境 | 解读 |
|------|------|
| 黄体期第 20 天，体温 36.9°C | 完全正常，安抚用户 |
| 超过预计月经日期后持续高温 | 可能妊娠或月经推迟，建议关注 |
| 月经期间体温偏高 | 结合其他症状评估是否存在感染 |
| 整个周期体温无明显双相变化 | 可能无排卵，建议妇科咨询 |

### 影响体温的因素

- **运动**：高强度运动后体温暂时升高（正常）
- **时间段**：清晨自然偏低，午后偏高（自然波动 0.3–0.5°C）
- **睡眠不足**：可能导致体温轻微升高
- **压力/皮质醇**：慢性压力可引起持续性低热
- **脱水**：可能引起轻微升高
- **酒精**：引起血管扩张，影响测量值
- **环境温度**：热水浴、高温环境会影响读数

### 跨域分析

- **体温 + 睡眠**：睡眠期间体温升高 → 可能在对抗感染或处于压力状态
- **体温 + 月经周期**：黄体期偏高是正常现象，不需担心
- **体温 + 运动**：运动后升高是暂时性正常现象；休息日持续偏高则需调查其他原因
- **体温 + 压力**：慢性压力 → 皮质醇 → 持续性低热，结合 HRV 交叉验证

### 沟通原则

先看背景，再看数字：

- **不好**："你的体温是 37.4°C，略高于正常。"
- **好**："你今天体温 37.4°C — 比你的个人基线 36.8°C 高了约 0.6°C，属于低热水平。我注意到你昨晚只睡了 5 小时，且压力指数偏高。休息和多喝水应该会有帮助。如果超过 24 小时未改善或升至 38°C 以上，建议看医生。"

### 红线

| 信号 | 处理方式 |
|------|---------|
| 体温 > 38°C 持续超过 2 天 | 建议就医排除感染 |
| 任何时候体温 > 39°C | 建议今天就看医生 |
| 持续 < 36.0°C | 可能提示代谢问题，下次体检时提及 |
| 发热伴严重头痛、颈部僵硬、皮疹或呼吸困难 | 立即就医 |
