# 血糖数据 CLI 获取指南

## 时间查询规范

- **无时间词** → 不传参数（工具默认今天）
- **"最近" / "这几天"** → 最近 7 天
- **"本周"** → 本周一至今天；**"上周"** → 上周完整自然周
- **"本月"** → 当月 1 日至今天；**"上个月"** → 上个完整自然月
- **"最近 X 天/周/月"** → 对应范围向前推算
- **"至今" / "历史"** → 两年前至今
- 查"上次何时"：先查最近 1 个月 → 无数据扩至 3 个月 → 最多 1 年

**日期准确性（不可违反）**：查到哪天说哪天，不可将 A 日期的数据说成 B 日期；部分日期无数据时如实说明，不得用相邻日期数据替代。

## 命令示例

### 获取今日血糖
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_glucose --date today
```

### 获取指定日期血糖
```bash
node ./skills/xiaoyi-health/bin/pha-claw.js get_blood_glucose --date 2024-01-15
```

> **注意**：血糖暂不支持日期范围查询，仅支持单日查询。如需趋势分析，可多次查询不同日期。

## 返回字段说明

| 字段 | 说明 |
|------|------|
| `value` | 血糖值（mmol/L） |
| `readings[]` | 全天各时段测量记录 |

## 参考范围（空腹）

| 状态 | 血糖值 |
|------|--------|
| 正常 | 3.9–6.1 mmol/L |
| 糖耐量异常 | 6.1–7.0 mmol/L |
| 糖尿病（需复查确认） | ≥7.0 mmol/L |

## 分析框架

### 血糖分级

| 指标 | 正常 | 糖尿病前期 | 糖尿病 |
|------|------|-----------|--------|
| 空腹血糖 (mmol/L) | 3.9-6.1 | 6.1-7.0 | ≥ 7.0 |
| 餐后 2 小时 (mmol/L) | < 7.8 | 7.8-11.1 | ≥ 11.1 |

**核心原则**：糖尿病前期通过生活方式改变是**可逆的**，这是关键干预窗口期。

### 控糖策略

**进食顺序效应（有循证支持）**：
1. 先吃蔬菜
2. 再吃蛋白质和脂肪
3. 最后吃碳水化合物
→ 可减少餐后血糖飙升 20-30%

**餐后步行**（最有效的单一干预）：
- 餐后 30 分钟内步行 15-30 分钟，可减少餐后血糖飙升 1-2 mmol/L

### 低血糖识别

| 血糖水平 | 严重程度 | 症状 |
|----------|---------|------|
| 3.9-3.3 mmol/L | 轻度 | 颤抖、出汗、饥饿、焦虑 |
| 3.3-2.8 mmol/L | 中度 | 意识混乱、视力模糊 |
| < 2.8 mmol/L | 重度 | 意识丧失 — 紧急情况 |

**15 法则**（轻中度低血糖）：摄入 15g 快速碳水，等待 15 分钟，复查血糖。

### 跨域分析

- **血糖 + 运动**：有氧运动即时降糖；抗阻训练改善长期胰岛素敏感性
- **血糖 + 睡眠**：睡眠不足（<6 小时）显著增加胰岛素抵抗
- **血糖 + 压力**：长期压力 → 皮质醇 → 空腹血糖升高

### 红线

| 信号 | 处理方式 |
|------|---------|
| 空腹血糖持续 ≥ 7.0 mmol/L | 建议就医评估糖尿病诊断 |
| 餐后血糖 > 11.1 mmol/L | 糖尿病范围，建议就医沟通 |
| 低血糖症状（颤抖、意识模糊） | 立即摄入快速糖分；15 分钟未改善则就医 |
| 重度低血糖（< 2.8 或意识丧失） | 医疗紧急情况，立即拨打急救电话 |
