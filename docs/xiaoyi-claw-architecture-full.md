### 双层协作流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           记忆协作流程                                       │
└─────────────────────────────────────────────────────────────────────────────┘

1. 【自动捕获】腾讯云插件 → L0 对话存储
   │
   ▼
2. 【自动提取】腾讯云插件 → L1 结构化记忆 + 向量化
   │
   ▼
3. 【冲突检测】yaoyao-memory-v2 → conflict_detector.py
   │  - 检测用户画像/事实/偏好/决策/系统冲突
   │  - 自动解决或提示用户确认
   ▼
4. 【知识分类】yaoyao-memory-v2 → knowledge_ops.py
   │  - 自动打标签 (decision/error/learning...)
   │  - 计算重要性等级 (Critical/High/Normal/Low)
   ▼
5. 【场景归纳】腾讯云插件 → L2 场景块
   │
   ▼
6. 【用户画像】腾讯云插件 → L3 persona.md
   │
   ▼
7. 【推送通知】yaoyao-memory-v2 → today-task / MeoW
   │  - 任务完成推送负一屏
   │  - 重要决策推送 MeoW
   ▼
8. 【云端备份】智能备份管理器 → 华为云盘 + IMA
   │  - 事件驱动，无需心跳
   │  - 华为云盘: 全量备份 (≥7天 或 ≥10次变更)
   │  - IMA: 增量同步 (≥24小时 或 重要决策)
   ▼
9. 【变更追踪】smart_backup.py → backup-state.json
      - 记录变更次数
      - 追踪重要决策
      - 自动触发备份
```

## 🎯 设计理念

1. **双层记忆** — 底层存储 + 上层增强，各司其职
2. **云端双备** — 华为云盘全量 + IMA 增量，双重保障
3. **事件驱动** — 无需心跳，关键操作自动触发备份
4. **不瞎编** — 多层验证，不确定就说不确定
5. **记得住** — 三级内存 + 情感驱动，重要的事忘不了
6. **想得深** — 8 种思考技能，不是简单问答
7. **跑得快** — 投机解码 + 多级缓存，响应快
8. **能多模态** — 能看图、能画图、能可视化
9. **懂语言** — NLP 模块提供底层语言理解能力
10. **可调用** — 统一入口 + 心跳脚本，OpenClaw 可直接调用
11. **能推送** — 双渠道推送，任务完成自动通知
