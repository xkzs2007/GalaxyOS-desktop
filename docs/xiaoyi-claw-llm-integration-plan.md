## 二、集成任务清单

### Phase 1: 核心记忆层集成 (2 周)

#### 1.1 memory-tencentdb + llm-memory-integration 集成
**目标**: 统一向量检索接口

**任务**:
- [ ] 分析 memory-tencentdb 的向量存储格式
- [ ] 分析 llm-memory-integration 的向量 API
- [ ] 设计统一向量检索接口
- [ ] 实现 `UnifiedVectorStore` 类
- [ ] 测试向量检索性能

**产出**:
- `src/integration/unified_vector_store.py`
- 向量检索性能报告

#### 1.2 memory-tencentdb + ontology 集成
**目标**: 记忆与知识图谱关联

**任务**:
- [ ] 分析 ontology 的图结构
- [ ] 设计 L1 记忆 → 实体提取流程
- [ ] 实现记忆到知识图谱的自动映射
- [ ] 实现图谱查询结果注入记忆检索
- [ ] 测试关联查询

**产出**:
- `src/integration/memory_ontology_bridge.py`
- 知识图谱关联测试报告

#### 1.3 memory-tencentdb + 2nd-brain 集成
**目标**: 个人知识库与记忆系统联动

**任务**:
- [ ] 分析 2nd-brain 的数据结构
- [ ] 设计 brain entry → L1 记忆同步机制
- [ ] 实现双向同步
- [ ] 测试知识库检索增强

**产出**:
- `src/integration/brain_memory_sync.py`
- 知识库同步测试报告

---

### Phase 2: 多模态能力集成 (1 周)

#### 2.1 图像理解 → 记忆存储
**目标**: 图像理解结果持久化

**任务**:
- [ ] 分析 xiaoyi-image-understanding 输出格式
- [ ] 设计图像记忆数据结构
- [ ] 实现图像嵌入存储
- [ ] 实现图像记忆检索
- [ ] 测试多模态检索

**产出**:
- `src/integration/multimodal_memory.py`
- 多模态检索测试报告

#### 2.2 图像生成 → 知识库关联
**目标**: 生成图像与知识库关联

**任务**:
- [ ] 分析 seedream-image_gen 输出
- [ ] 实现生成图像自动归档到 brain
- [ ] 实现图像与实体的关联
- [ ] 测试图像知识库查询

**产出**:
- `src/integration/image_brain_link.py`

---

### Phase 3: 智能记忆管理 (1 周)

#### 3.1 重要性评分算法
**目标**: 为智能遗忘提供依据

**任务**:
- [ ] 设计重要性评分维度
  - 访问频率
  - 时间衰减
  - 情感权重
  - 用户反馈
- [ ] 实现评分算法
- [ ] 集成到 memory-tencentdb
- [ ] 测试评分准确性

**产出**:
- `src/memory/importance_scorer.py`
- 重要性评分测试报告

#### 3.2 智能遗忘机制
**目标**: 自动清理低价值记忆

**任务**:
- [ ] 设计遗忘策略
  - 低重要性 → 归档
  - 过期记忆 → 删除
  - 冲突记忆 → 合并
- [ ] 实现遗忘调度器
- [ ] 集成到 memory-tencentdb 清理流程
- [ ] 测试遗忘效果

**产出**:
- `src/memory/smart_forgetter.py`
- 遗忘策略测试报告

#### 3.3 记忆反思模块（新增）
**目标**: 让 AI 具备元认知能力，主动发现和修正问题

**任务**:
- [ ] 设计反思记录数据结构
- [ ] 实现错误检测机制
  - 命令执行失败
  - 用户纠正识别
  - 自我发现知识过时
- [ ] 实现模式识别算法
  - 重复错误检测（≥3次触发反思）
  - 相似场景失败分析
- [ ] 实现自动改进机制
  - 生成改进建议
  - 规则自动更新（高优先级）
  - 用户确认机制（低优先级）
- [ ] 集成到 HEARTBEAT.md
- [ ] 测试反思闭环

**产出**:
- `src/memory/memory_reflector.py`
- `.learnings/REFLECTIONS.jsonl` 存储结构
- 反思模块测试报告

**详细设计**: 见 `memory/memory-reflection-module-design.md`

- [ ] 编写 API 文档
