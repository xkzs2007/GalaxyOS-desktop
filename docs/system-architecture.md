```
## 🔤 Layer 2: NLP 处理层

```

│                    (nlp_processor.py)                           │

│  │  5. 文本摘要 (TextSummarizer)                            │    │
│  │     ├─ 分句                                              │    │

│  │     └─ 关键句提取                                        │    │
│  │                                                         │    │
│  │  6. 语义相似度 (SemanticSimilarity)                      │    │

│  │  └─ calculate_memory_importance() → 重要性计算           │    │

│  │  知识图谱接口:                                           │    │
│  │  └─ extract_triples() → 三元组提取                      │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                 │
│  性能特点:                                                       │
│  ├─ 首次加载: ~600ms (jieba 模型)                               │
│  ├─ 后续处理: <1ms                                              │
│  ├─ 缓存支持: LRU 缓存                                          │
│  └─ 单例模式: 全局唯一实例                                      │
│                                                                 │
